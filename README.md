# STARVIVORS

Initial Phaser 3, TypeScript, and Vite browser scaffold for STARVIVORS.

## Install

```powershell
npm install
```

## Dev

```powershell
npm run dev
```

Open the local URL printed by Vite.

## Build

```powershell
npm run build
```

## Current Scope

This scaffold renders a dark tactical sci-fi starfield, a placeholder Interceptor-style triangle ship, a 9 by 9 viewport arena, a following camera, and debug text.

No player movement, enemies, weapons, black holes, upgrades, shop logic, audio, Electron, or Steam integration has been added.
